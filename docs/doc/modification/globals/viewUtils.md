# 视图相关工具

## viewUtils.findParentById(view, id)

-   `view` View
-   `id` int
-   `return` View

通过 id 寻找视图的父视图

## viewUtils.dpToPx(dp)

-   `dp` int
-   `return` int

dp 转 px

## viewUtils.pxToSp(px)

-   `px` floaty
-   `return` float

px 转 sp

## viewUtils.pxToDp(px)

-   `px` int
-   `return` int

px 转 dp

## viewUtils.spToPx(sp)

-   `sp` float
-   `return` int

sp 转 px
